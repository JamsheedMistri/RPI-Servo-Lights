# RPI-Servo-Lights
This repository holds the small and simple C program that turns a servo to flip a swtich on and off given an argument. The servo is connected to the Raspberry Pi GPIO pin.

## Building
```bash
sh /path/to/build.sh
```

## Running
```bash
sudo /path/to/run <args>
```
**Allowed Arguments**
* 0 — off
* 1 — on
